Fake Food Detector

This project helps users identify counterfeit food products by analyzing multiple images of the product packaging using machine learning and OCR techniques. Upload front/back/side/barcode images and check authenticity.

How it works

  • Compare packaging features using a trained image model
  • Extract and validate text from labels using OCR
  • Combine multi-view signals into a final authenticity score
